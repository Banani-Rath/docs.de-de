# Compiler Error CS0001
Interner Compilerfehler  
  
 Ermitteln Sie, ob der Compilerfehler auftritt, weil der Compiler eine unerwartete Syntax nicht analysieren kann.  Wenn dieser Fehler wiederholt ausgegeben wird, wenden Sie sich an Microsoft.
