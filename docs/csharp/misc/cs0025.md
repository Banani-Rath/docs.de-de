# Compilerfehler CS0025
Die Standardbibliotheksdatei 'datei' wurde nicht gefunden.  
  
 Eine Datei, die vom Compiler benötigt wird, wurde nicht gefunden. Überprüfen Sie, ob der Pfad richtig und die Datei vorhanden ist.  
  
 Wenn die Datei eine [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]\-Systemdatei ist, müssen Sie möglicherweise Ihre [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]\-Installation reparieren oder von Grund auf neu installieren.
