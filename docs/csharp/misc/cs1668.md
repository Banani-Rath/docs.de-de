# Compilerwarnung (Stufe 2) CS1668
Ungültiger Suchpfad 'Pfad' in 'Pfadzeichenfolge' angegeben – 'Systemfehlermeldung'  
  
 Der für [\/lib](../../csharp/language-reference/compiler-options/lib-compiler-option.md) an der Befehlszeile angegebene Pfad oder ein Pfad in der LIB\-Umgebungsvariablen ist ungültig. Überprüfen Sie den verwendeten Pfad, um sicherzustellen, dass er vorhanden und zugänglich ist. Die Fehlermeldung in einfachen Anführungszeichen gibt den vom Betriebssystem zurückgegebenen Fehler an.
