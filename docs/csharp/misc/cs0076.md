# Compilerfehler CS0076
Der Enumeratorname "value\_\_" ist reserviert und kann nicht verwendet werden.  
  
 Ein Element in einer [Enumeration](../../csharp/language-reference/keywords/enum.md) darf keinen Bezeichner mit dem Namen **value\_\_** aufweisen.
