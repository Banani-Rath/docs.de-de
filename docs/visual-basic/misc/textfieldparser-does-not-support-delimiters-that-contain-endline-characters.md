# TextFieldParser unterst&#252;tzt keine Trennzeichen, die Zeilenendzeichen enthalten
Es wurde ein Trennzeichen angegeben, das ein Zeilenendezeichen enthält, aber der `TextFieldParse`r unterstützt keine Trennzeichen, die Zeilenendezeichen enthalten.  
  
### So beheben Sie diesen Fehler  
  
-   Geben Sie ein ordnungsgemäßes Trennzeichen an.
