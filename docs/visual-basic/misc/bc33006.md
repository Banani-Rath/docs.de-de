# &#39;End Operator&#39; muss die erste Anweisung in der Zeile sein
Einem Doppelpunkt \(`:`\) als Trennzeichen folgt eine `End Operator`\-Anweisung.`End Operator` muss die einzige Anweisung der jeweiligen Quellzeile sein.  
  
 **Fehler\-ID:** BC33006  
  
### So beheben Sie diesen Fehler  
  
-   Platzieren Sie jede Anweisung in einer eigenen Zeile.
