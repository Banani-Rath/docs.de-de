# Der Vorgang wurde durch den Benutzer abgebrochen
Ein Benutzer hat STRG\+UNTBR oder eine anderen Taste für Unterbrechen gedrückt.  
  
### So beheben Sie diesen Fehler  
  
-   Wiederholen Sie den Vorgang.
