# &#39;&lt;typname&gt;&#39; ist ein Typ in &#39;&lt;containername&gt;&#39; und kann nicht als Ausdruck verwendet werden
Einer der verwendeten Namen wurde bereits als Name eines Typs deklariert und kann nicht in einem Ausdruck verwendet werden.  
  
 **Fehler\-ID:** BC30691  
  
### So beheben Sie diesen Fehler  
  
-   Geben Sie dem Member einen eindeutigen Namen.  
  
-   Schlie√üen Sie den Namen des Members in eckige Klammern ein.
