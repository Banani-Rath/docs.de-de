# &quot;Is&quot; erwartet
Ein `TypeOf`\-Operator tritt ohne eine `Is`\-Klausel auf.  
  
 **Fehler\-ID:** BC30224  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie nach dem `TypeOf`\-Operator eine `Is`\-Klausel hinzu, z. B. `TypeOf MyControl Is ComboBox`.
