# Das Argument &lt;Argumentname&gt; muss gr&#246;&#223;er oder gleich -1 sein.
Der Wert eines Arguments ist kleiner als \-1.  
  
### So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass der Argumentwert innerhalb des gültigen Bereichs liegt.  
  
2.  Wenn das Argument durch eine Berechnung bereitgestellt wird, überprüfen Sie die von der Berechnung verarbeiteten Werte.
