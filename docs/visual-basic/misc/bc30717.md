# Try-Anweisungen sind im Direktfenster nicht g&#252;ltig.
`Try`\-Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler\-ID:** BC30717  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Try`\-Anweisungen aus Debugcode.
