# Typen sind in diesem Kontext nicht vorhanden.
Sie haben versucht, auf einen Typ in einer `#If`\-Direktive zu verweisen.  
  
 **Fehler\-ID:** BC30458  
  
### So beheben Sie diesen Fehler  
  
1.  Entfernen Sie den Verweis.
