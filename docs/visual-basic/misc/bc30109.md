# &quot;&lt;Typname&gt;&quot; ist ein Klassentyp und kann nicht als Ausdruck verwendet werden.
Ein Typname tritt an einer Stelle auf, an der ein Ausdruck erforderlich ist. Ein Ausdruck muss aus einer Kombination aus Variablen, Konstanten, Literalen, Eigenschaften und `Function`\-Prozeduraufrufen bestehen.  
  
 **Fehler\-ID:** BC30109
