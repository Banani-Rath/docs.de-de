# Umwandeln in &lt;Typ&gt; nicht m&#246;glich.
Es wurde versucht, eine ungültige Konvertierung vorzunehmen.  
  
 **Fehler\-ID:** BC30748  
  
### So beheben Sie diesen Fehler  
  
-   Wählen Sie einen kompatiblen Konvertierungstyp.
