# Case-Anweisungen sind im Direktfenster nicht g&#252;ltig.
`Case`\-Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler\-ID:** BC30719  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Case`\-Anweisungen aus dem **Direktfenster**.
