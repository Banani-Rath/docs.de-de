# Komma oder &#39;)&#39; erwartet (Typargumentliste)
Eine Typargumentliste enthält ein unerwartetes Element. Trennen Sie Typargumente in einer solchen Liste mit Kommas \(`,`\), und beenden Sie die Liste mit einer schließenden Klammer \(`)`\).  
  
 **Fehler\-ID:** BC32094  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie ein Komma hinzu, und setzen Sie die Erstellung der Typargumentliste fort, oder fügen Sie eine schließende Klammer hinzu, um die Liste zu beenden.
