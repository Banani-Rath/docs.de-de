# &#39;&lt;name1&gt;&#39; is ambiguous, imported from the namespaces or types &#39;&lt;name2&gt;&#39;
Sie haben einen Namen angegeben, der mehrdeutig ist und dadurch einen Konflikt mit einem anderen Namen verursacht.  Für den [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]\-Compiler wurden keine Regeln zur Konfliktlösung implementiert, sodass Sie die Mehrdeutigkeit des Namens selbst auflösen müssen.  
  
 **Fehler\-ID:** BC30561  
  
### So beheben Sie diesen Fehler  
  
1.  Machen Sie den Namen eindeutig, indem Sie Namespaceimporte entfernen.  
  
2.  Kennzeichnen Sie den Namen vollständig.
